# Framing - Para Amantes da Fotografia

## Projeto

Framing é um projeto acadêmico individual da faculdade de Tecnologia São Paulo Tech School (SPTECH), do curso de Ciência da Computação, 1º semestre.

Este projeto é uma rede social em pequena escala, voltada para quem ama fotografia e gosta de compartilhar seus registros e emoções

# Instalação

Você pode clonar este repositório e executar em ambiente local, sendo necessário alguns dependências:

- `npm`
